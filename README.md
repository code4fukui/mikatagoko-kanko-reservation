# mikatagoko-kanko-reservation

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

This is a web application that displays the accommodation reservation status in the Mikata-goko area in an easy-to-understand graph.

## Demo
<https://code4fukui.github.io/mikatagoko-kanko-reservation/>

## Features
- Displays accommodation reservation status in the Mikata-goko area
- Provides data on occupancy rate (OCC), average daily rate (ADR), and revenue per available room (RevPAR)
- Allows users to select a date range to view the data

## Data / API
The data is sourced from the "FTAS" (Fukui Tourism Analysis System) provided by the Fukui Prefecture Tourism Federation. For other areas, please refer to <https://www.fuku-e.com/FTAS> section `1-3)`.
