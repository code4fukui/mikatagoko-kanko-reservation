# 三方五湖エリア 宿泊予約状況

三方五湖エリアの宿泊予約状況を、グラフで見やすく表示するウェブアプリケーションです。福井県観光連盟提供のオープンデータを使用しています。

## デモ
[三方五湖エリア 宿泊予約状況](https://code4fukui.github.io/mikatagoko-kanko-reservation/)

## 機能
- 三方五湖エリアの宿泊予約状況をグラフで表示
- 利用総人数、室数、平均泊数、利用金額合計などの指標を確認可能
- 客室稼働率(OCC)、客室平均単価(ADR)、RevPAR(OCC x ADR)などの指標を表示
- 日付範囲を選択して表示できる

## データ・API
- 宿泊予約データ: [福井県観光データ分析システム「FTAS」](https://www.fuku-e.com/feature/detail_266.html)
- ホテル情報: 福井県観光連盟提供
